## <a name="intro">🤖 はじめに</a>

薬の飲み忘れを防ぎ、薬の継続を習慣化するスマホアプリ、medareco を紹介します。

EVANS は自由にチャンネルが作成でき、グループごとに会話ができるアプリです。
